# Bad Icecream
В данном проекте реализован аналог игры "Bad Icecream".

## Техническое задание
Необходимо выполнить курсовую работу в соответствии с учебным пособием [Беляев С.А. «Разработка игр на языке JavaScript»](./Razrabotka_igr_na_JS.pdf)

Все делают в соответствии с общим заданием в соответствии с учебным пособием на «чистом» JavaScript (ES6). В группах по 1 человеку.

- Минимум 2 уровня игры
- Реализованы все менеджеры в соответствии с учебным пособием (УП)
- Есть таблица рекордов
- Есть препятствия
- Есть «интеллектуальные» противники и «бонусы»
- Используются tiles с редактором [Tiled](https://www.mapeditor.org/) в соответствии с УП
- Изображения объединяются в атлас в формате JSON, например, в редакторе [Leshy SpriteSheet Tool](https://www.leshylabs.com/apps/sstool/)
- Спрайты можно брать, например, [тут](https://itch.io/game-assets/free)
